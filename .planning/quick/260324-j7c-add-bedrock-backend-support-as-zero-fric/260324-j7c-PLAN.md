       - LLM_BACKEND: str = os.getenv("LLM_BACKEND", "anthropic")  # "anthropic" | "bedrock"
       - AWS_REGION: str = os.getenv("AWS_REGION", "us-east-1")
       - AWS_ACCESS_KEY_ID: str = os.getenv("AWS_ACCESS_KEY_ID", "")
       - AWS_SECRET_ACCESS_KEY: str = os.getenv("AWS_SECRET_ACCESS_KEY", "")

    3. Create sentinel/llm_client.py:
       - Import AsyncAnthropic from anthropic, AsyncAnthropicBedrock from anthropic (the bedrock extra makes this available)
       - Import get_settings from sentinel.config
       - Define _MODELS dict with per-backend defaults:
         "anthropic": {"supervisor": "claude-opus-4-6", "agent": "claude-sonnet-4-6"}
         "bedrock": {"supervisor": "us.anthropic.claude-opus-4-5-20251101-v1:0", "agent": "us.anthropic.claude-sonnet-4-5-20251001-v1:0"}
       - get_async_client() -> AsyncAnthropic | AsyncAnthropicBedrock:
         reads settings.LLM_BACKEND
         if "anthropic": return AsyncAnthropic(api_key=settings.ANTHROPIC_API_KEY)
         if "bedrock": return AsyncAnthropicBedrock(aws_region=settings.AWS_REGION, aws_access_key=settings.AWS_ACCESS_KEY_ID, aws_secret_key=settings.AWS_SECRET_ACCESS_KEY)
         else: raise ValueError(f"Unknown LLM_BACKEND: {settings.LLM_BACKEND}")
       - get_model_ids() -> dict[str, str]:
         reads settings.LLM_BACKEND, settings.SUPERVISOR_MODEL, settings.AGENT_MODEL
         starts with _MODELS[backend] defaults
         overrides supervisor if SUPERVISOR_MODEL is non-empty
         overrides agent if AGENT_MODEL is non-empty
         returns {"supervisor": ..., "agent": ...}

    4. In .env.example: append after existing vars:
       ```
       # --- LLM Backend ---

       LLM_BACKEND=anthropic
       # AWS_REGION=us-east-1
       # AWS_ACCESS_KEY_ID=
       # AWS_SECRET_ACCESS_KEY=
       # Optional: override model IDs (leave blank for backend defaults)
       # SUPERVISOR_MODEL=
       # AGENT_MODEL=
       ```
