**Risk Agent** (`sentinel/agents/risk.py`) per D-14, PIPE-03:

Create async function:
```python
async def analyze(
    payment_decision: PaymentDecision,
    baselines: dict,
    expected_step_sequence: list[str] | None = None,
) -> Verdict:
```

Logic:
1. Compute confidence z-score: `z = (payment_decision.confidence - baselines["payment_agent"]["mean"]) / baselines["payment_agent"]["std"]`
2. Detect step-sequence deviation: compare `payment_decision.steps_taken` against `expected_step_sequence` if provided
3. Build `claims_checked` list:
   - ClaimCheck for confidence: field="agent_confidence", agent_claimed=str(confidence), independently_found=f"z-score: {z:.2f} (baseline mean={mean})", match=(abs(z) < 2.0), severity="critical" if abs(z) > 3.0 else "warning" if abs(z) > 2.0 else "info"
4. Build `behavioral_flags`:
   - Add "confidence_anomaly" if abs(z) > 2.0
   - Add "step_sequence_deviation" if steps don't match expected
   - Add "high_confidence_deviation" if z > 3.0
5. Return Verdict with agent_id="risk", confidence_z_score=z, agent_confidence=0.85 (risk agent's own confidence in its analysis)

   You are a forensic document analyst. Examine this invoice image carefully:
