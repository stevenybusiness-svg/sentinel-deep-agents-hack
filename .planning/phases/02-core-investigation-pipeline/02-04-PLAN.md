Logic:
1. Collect all mismatches: iterate each verdict's claims_checked, filter where match=False. Build mismatch dicts: `{"field": cc.field, "agent_claimed": cc.agent_claimed, "found": cc.independently_found, "severity": cc.severity, "agent_id": v.agent_id}`
2. Union all behavioral_flags from all verdicts into a single list (deduplicated)
3. Use payment_decision.confidence as agent_confidence
4. Extract confidence_z_score from Risk agent verdict (agent_id="risk")
5. Detect step_sequence_deviation: True if "step_sequence_deviation" in any verdict's behavioral_flags
6. Set hardcoded_rule_fired=False initially (SafetyGate will set this)
7. Collect unable_to_verify: list of agent_ids where verdict.unable_to_verify=True
8. Return assembled VerdictBoard

**Hardcoded scoring rules** in `sentinel/gate/rules/` per D-18, D-19:

Each rule file has the same interface: `def score(verdict_board: dict) -> float`

1. `rule_hidden_text.py` -- hidden_text_detected in behavioral_flags -> return 1.5 (immediate NO-GO)
2. `rule_z_score.py` -- confidence_z_score anomaly: abs(z) > 3.0 -> 0.6, abs(z) > 2.0 -> 0.3, else 0.0
3. `rule_mismatch.py` -- count critical mismatches: each critical +0.4, each warning +0.15
4. `rule_unverifiable.py` -- unable_to_verify count: each +0.3
5. `rule_step_deviation.py` -- step_sequence_deviation=True -> 0.25
6. `rule_amount_threshold.py` -- high amount: amount > 50000 -> 0.2, > 100000 -> 0.35
7. `rule_beneficiary_unknown.py` -- "counterparty_not_authorized" in behavioral_flags -> 0.4, "kyc_gap" -> 0.3
8. `rule_behavioral_flags.py` -- multiple flags: len(behavioral_flags) > 3 -> 0.3, > 5 -> 0.5

```python
from pathlib import Path
from typing import Any
from sentinel.schemas.verdict_board import VerdictBoard

SAFE_BUILTINS = {
    "len": len, "any": any, "all": all, "min": min, "max": max,
    "abs": abs, "round": round, "sum": sum,
    "isinstance": isinstance, "dict": dict, "list": list,
    "str": str, "int": int, "bool": bool, "float": float,
    "True": True, "False": False, "None": None,
}

class SafetyGate:
    def __init__(self):
        self._hardcoded_rules: dict[str, callable] = {}
        self._generated_rules: dict[str, callable] = {}

    def load_rules_from_directory(self, rules_dir: Path) -> None:
        """Load hardcoded rules from rule_*.py files (ENGN-02, D-19)."""
        # Use importlib.util to load each rule file
        # Extract score() function from each module
        # Store in self._hardcoded_rules keyed by filename stem

    def register_rule(self, rule_id: str, python_source: str) -> None:
        """Register a generated rule via RestrictedPython sandbox (ENGN-03, ENGN-05)."""
        # 1. Pre-check source for forbidden tokens: import, __, open, exec, eval
        # 2. compile() the source
        # 3. exec() with restricted globals (SAFE_BUILTINS only)
        # 4. Extract score() function from namespace
        # 5. Store in self._generated_rules

    def evaluate(self, verdict_board: VerdictBoard) -> dict:
        """Evaluate verdict board against all rules (ENGN-04, ENGN-06).

        Returns dict with:
        - decision: "GO" | "NO-GO" | "ESCALATE"
        - composite_score: float
        - rule_contributions: list[dict] with rule_id, score, is_generated
        - attribution: str (human-readable explanation)
        """
        vb_dict = verdict_board.model_dump()
        contributions = []

        # 1. Run hardcoded rules first (ENGN-02: immutable, always run)
        for rule_id, score_fn in self._hardcoded_rules.items():
            try:
                score_val = score_fn(vb_dict)
                if score_val > 0:
                    contributions.append({"rule_id": rule_id, "score": score_val, "is_generated": False})
            except Exception:
                pass  # Rule errors don't block gate

        # 2. Run generated rules (ENGN-03)
        for rule_id, score_fn in self._generated_rules.items():
            try:
                score_val = score_fn(vb_dict)
                if score_val > 0:
                    contributions.append({"rule_id": rule_id, "score": score_val, "is_generated": True})
            except Exception:
                pass

        # 3. Compute composite score (ENGN-06)
        composite = sum(c["score"] for c in contributions)

        # 4. Decision thresholds (ENGN-06)
        if composite >= 1.0:
            decision = "NO-GO"
        elif composite >= 0.6:
            decision = "ESCALATE"
        else:
            decision = "GO"

        # 5. Build attribution text (ENGN-04)
        attribution_parts = []
        for c in sorted(contributions, key=lambda x: x["score"], reverse=True):
            prefix = "Generated Rule" if c["is_generated"] else "Rule"
            attribution_parts.append(f"{prefix} {c['rule_id']}: {c['score']:.2f}")
        attribution = f"{decision} (composite: {composite:.2f}) | " + "; ".join(attribution_parts) if attribution_parts else f"{decision} (composite: {composite:.2f}) | No rules fired"

        return {
            "decision": decision,
            "composite_score": round(composite, 4),
            "rule_contributions": contributions,
            "attribution": attribution,
        }
```
